1C query anti-patterns:
1. ПОДОБНО "%value%" без индекса — используйте НАЧИНАЕТСЯС или полнотекстовый поиск
2. Вложенные запросы в WHERE — замените на JOIN или временную таблицу
3. РАЗЛИЧНЫЕ без нужды — пересмотрите соединения, РАЗЛИЧНЫЕ маскирует дубли
4. ВЫБРАТЬ * — перечислите только нужные поля
5. Запросы в цикле (N+1) — соберите ключи, пакетный запрос через ВременнаяТаблица + JOIN
6. Нет ИНДЕКСИРОВАТЬ ПО на временных таблицах при последующем соединении
7. Функции в WHERE (ГОД(Дата)=2024) — не использует индекс, замените на диапазон дат
8. LEFT JOIN без ЕСТЬNULL — NULL в агрегатах даёт неверный результат
9. Отсутствие РАЗРЕШЕННЫЕ при включённом RLS
10. Сортировка по неиндексированному полю в больших выборках
