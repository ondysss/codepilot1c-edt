# CodePilot1C EDT — hotfix: стабильное редактирование файлов и tool loop

Сборка: `0.1.7.20260428-0404`

Хотфикс к релизу `0.1.7.20260421-0335`. В сборку включены исправления для сценария редактирования файлов через агента, профилирования EDT и отображения статистики токенов.

---

## Стабильность редактирования файлов

- Исправлен порядок обработки `edit_file`: агент получает результат выполненного инструмента до дополнительных подсказок против повторения действий.
- Это устраняет сценарий, в котором агент мог повторять одно и то же редактирование, не увидев результат предыдущего вызова инструмента.

---

## Профилирование EDT

- Инструменты `start_profiling` и `get_profiling_results` доступны в рабочих профилях агента, где используются EDT debug и YAxUnit-сценарии.
- Для запуска/остановки профилирования требуется разрешение пользователя, просмотр результатов разрешен без дополнительного подтверждения.

---

## Статистика токенов

- При преобразовании обычного ответа провайдера в потоковый вывод чата сохраняется реальная статистика токенов.
- Нижняя панель чата теперь получает данные провайдера вместо расчетной оценки в сценариях backend fallback.
